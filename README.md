# howtools

> 好工具，一个力求简单的js工具库

# 安装方式

`npm i howtools`

# 简要文档说明

> 简单介绍函数作用，具体需要在`src`文件夹下查看函数的具体作用。

## 数据操作类
### src/data/array
  - `columnData` 获取对象数组某一列数据
  - `rows2columns` 行转列函数
  - `unique` 数组去重，支持对象数组/一般数组

### src/files/blob
  - `saveFileFromBlob` Blob流保存为文件
  - `blob2Json` Blob文件转json

### src/utils/util
  -  `debounce` 防抖函数
  -  `throttle` 节流函数
